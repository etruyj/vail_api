VAPIr (Vail API)

This script provides access to the Spectra Logic Vail Sphere via the API. Available options are:

--account	Specify the account name (username) or account ID of an AWS account to filter the results to just that account. Use "all" to list all accounts.
--active-only	Flag to list only user accounts activated in the Vail system.
--command(-c)	The command to be executed. Commands can be listed with -c help, -c help-basic, or -c help-advanced
--endpoint(-e)	IP Address of the Stack
--ignore-ssl	Flag to ignore the SSL certificate provided by the Sphere
--output-format	Specify the output format of the response. Default is table. Accepts csv, shell, table, and xml.
--password(-p)  Password
--username(-u)	Username
